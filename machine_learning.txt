Machine learning is a branch of artificial intelligence that enables computers to learn patterns from data and make predictions without being explicitly programmed. Instead of writing rules manually, developers provide algorithms with data, and the algorithms learn relationships within that data. Machine learning has become a key technology in many modern applications such as recommendation systems, fraud detection, medical diagnosis, and natural language processing. Machine learning models are generally divided into three categories: supervised learning, unsupervised learning, and reinforcement learning.

In supervised learning, the algorithm learns from labeled data where each input example has a corresponding output label. Common supervised learning tasks include classification and regression. For example, an email spam detection system learns to classify emails as spam or not spam based on labeled training data. Regression models predict continuous values such as house prices or stock market trends. Popular supervised learning algorithms include decision trees, support vector machines, and neural networks.

Unsupervised learning, on the other hand, deals with unlabeled data. The algorithm attempts to discover patterns, structures, or relationships within the data. Clustering is one common unsupervised learning task, where similar data points are grouped together. Dimensionality reduction is another technique used to reduce the number of variables in a dataset while preserving important information. Algorithms such as K-means clustering and principal component analysis are widely used for these tasks. Unsupervised learning is particularly useful for exploratory data analysis and pattern discovery.
